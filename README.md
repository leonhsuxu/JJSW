# JJSW
1车间日报表

界面采用了 “顶部标题 + 参数设置区 + 数据录入区 + 彩色报表区” 的四段式设计，层次分明，逻辑清晰

界面设计细节亮点：
大标题与署名：
“晶晶生物装袋车间” 占据顶部核心位置，使用深蓝色加粗大字，非常显眼。 
“by Leon” 位于标题右下方，采用灰色斜体小字，低调而专业。  
设置区锁定保护：  
初始锁定：默认情况下，所有的参数框都是灰色的（不可编辑）。  
点击解锁：点击解锁按钮后，按钮背景变橘色提醒，文字变为“保存锁定”，此时才可修改默认配方。  
彩色结果报表 (核心特色)：  
蓝色 (#1565C0)：用于装袋和生产总数。代表成品产出，给人稳重、达标的视觉感。  
红色 (#D32F2F)：用于不良、退回。红色代表预警，方便统计人员一眼盯住损耗。  
绿色 (#2E7D32)：用于所有原材料。代表农业/生物生产的自然属性，也方便与产量数据区分。  
等宽对齐：使用等宽字体，让每一行的冒号和数字都能完美对齐，方便截图后直接作为工作汇报发送。  
交互友好：  
当选择“方式2”手填时，输入框会自动跳出。  
计算按钮宽大，即使在车间戴着手套（只要是触控手套）也容易点中。
